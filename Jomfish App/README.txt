🚀 Installation:
Jomfish is installed under 'C:Program FilesJomfish' and automatically added to the PATH.
Once installed, you can launch Jomfish from the console with:

jomfish
arena

⚡ Example:

🛠️ Troubleshooting:
If 'jomfish' is not recognized, restart the PC or run the following command in CMD:

setx PATH "%PATH%; C:Program FilesJomfish" /M
